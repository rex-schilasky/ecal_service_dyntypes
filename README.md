# ecal_service_dyntypes
Prototype to check dynamic service type description

## How to build

1. CMake the project
	```console
	mkdir _build
	cd _build
	cmake ..
	```

2. Build the project
	- Linux: `make`
	- Windows: Open `_build\ecal_service_dyntypes.sln` with Visual Studio and build the example project
